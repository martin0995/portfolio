TMDB

The Movies Data Base (proyecto independiente)

TMDB es mi primer proyecto como desarrollador. TMDb (The Movie Database) es una base de datos en línea y completa de información relacionada con películas y programas de televisión. La aplicación incluye detalles de elenco y equipo, resúmenes de trama, calificaciones y reseñas. Además, la página proporciona información sobre fechas de lanzamiento, compañías de producción y otros datos relevantes.

El objetivo principal de TMDb es proporcionar información precisa y actualizada sobre películas y programas de televisión. A su vez, mi proyecto está conectao a la API de TMDB recolectando y actualizando toda la información más reciente.

Este es mi primer proyecto explorando la aplicación de todos mis conocimientos de programación y aprendiendo nuevas tecnologías. Todavía no se encuentra terminado pero espero que lo disfruten!
