TMDB

The Movies Data Base (solo project)

TMDB is my first project. The TMDb (The Movie Database) is a comprehensive online database of information related to movies and TV shows. The applictaion includes cast and crew details, plot summaries, ratings, and reviews. Additionally, the website provides information on release dates, production companies, and other relevant data.

TMDb's primary goal is to provide accurate and up-to-date information about movies and TV shows. My projcet is connected to the TMDB's API collecting all the respective information up to date.

This is my first projcet, exploring how to apply all my knowledge and also learning new technologies. It is not finished yet but I hope you like it!
