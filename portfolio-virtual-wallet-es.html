Billetera Virtual

Proyecto Profesional para Itesa | Innovation Hub

Desarrollamos una billetera virtual donde los usuarios pueden realizar transacciones facilmente en la Web 3.0. A su vez, la aplicación se conecta a una billetera externa, MetaMask, para mayor facilidad y seguridad de los activos.

Como equipo, emitimos nuestro propio token en la blockchain. El nombre del token es "TukiCoin" y actualmente opera dentro de la cadena de bloques Goerli.

El objetivo principal del proyecto es viralizar la implementación y el uso del "TukiCoin". Además, aplicamos la estrategia de negocio "get viral", donde se asignan recompensas a aquellos usuarios que invitan amigos a registrarse. También, hay un rank para ver quién invitó a más amigos.

Finalmente, contamos con una vista para administradores con distintas estadísticas (cantidad de tokens emitidos, número de usuarios registrados y más) y otras funcionalidades para administrar la aplicación. Espero que les guste!
