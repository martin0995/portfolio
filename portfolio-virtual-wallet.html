Virtual Wallet

Professional Project for Itesa | Innovation Hub

We developed a virtual wallet where users can easily make transactions in the Web 3.0. The appliation connects to an external wallet, MetaMask, to make transactions easier and secure.

As a team, we also issued our own token in the blockchain. The name of the token is "TukiCoin" and is currently operating within the Goerli blockchain.

The main purpose of the project is to viralize the implementation and usage of the "TukiCoin". Furthermore, we applied the "get viral" strategy, to reward users that invite friends to register in the app. There is also a rank to see who invited more friends, there are also rewards for that!

Finally, there is an admin view where different statistics can be displayed (amount of tokens issued, number of users, and more) and also other functionalities to manage the app.
